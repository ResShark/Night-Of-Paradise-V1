## A propos de esx_addons_gcphone
Permet de faire la liaison entre le téléphone et les métiers esx.

Veuillez mettre esx_addons_gcphone & gcphone avant les jobs.
Exemple :
```yml

  start mysql-async
  start essentialmode
  start esplugin_mysql
  start es_extended

  start esx_addons_gcphone
  start gcphone

  start esx_mecanojob
  start esx_job2
  start esx_job3
